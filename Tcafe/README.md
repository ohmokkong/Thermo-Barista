# ☕ Thermo-Barista (음료 냉각 예측 분석기)

**Thermo-Barista**는 뉴턴의 냉각 법칙(Newton's Law of Cooling)을 기반으로, 뜨거운 커피나 차가운 음료가 사용자가 원하는 **적정 온도(Golden Temperature)**에 도달하는 시간을 과학적으로 예측해주는 Python CLI 도구입니다.

단순한 수치 계산뿐만 아니라, 시간에 따른 온도 변화를 **그래프(Matplotlib)**로 시각화하여 직관적인 정보를 제공합니다.

## ✨ 주요 기능

- **골든 타임 예측**: 현재 음료 온도와 환경 온도를 기반으로 마시기 가장 좋은 온도에 도달하는 시간을 계산합니다.
- **냉각/가열 시뮬레이션**: 뜨거운 음료가 식는 경우뿐만 아니라, 차가운 음료가 미지근해지는 경우도 지원합니다.
- **데이터 시각화**: 온도 변화 곡선을 그래프로 그려주며, 골든 타임 지점을 시각적으로 표시합니다.
- **한글 지원**: Windows, Mac, Linux 등 OS에 맞춰 그래프의 한글 폰트 깨짐 현상을 자동으로 방지합니다.
- **사용자 친화적 가이드**: 용기 재질별 냉각 계수($k$) 추천 값을 제공합니다.

## 🛠️ 설치 방법 (Installation)

이 프로젝트는 Python 3.x 환경에서 실행됩니다.

### 1. 저장소 복제 (Clone)

```bash
git clone https://github.com/your-username/Thermo-Barista.git
cd Thermo-Barista
```

### 2. 가상환경 생성 및 활성화 (권장)

프로젝트의 독립적인 실행 환경을 위해 가상환경을 사용하는 것이 좋습니다.

- **Windows:**
  ```bash
  python -m venv .venv
  .venv\Scripts\activate
  ```
- **Mac/Linux:**
  ```bash
  python3 -m venv .venv
  source .venv/bin/activate
  ```

### 3. 필수 라이브러리 설치

```bash
pip install -r requirements.txt
```

## 🚀 사용 방법 (Usage)

터미널에서 아래 명령어로 프로그램을 실행합니다.

```bash
python main.py
```

프로그램이 실행되면 화면의 안내에 따라 다음 정보를 입력하세요:

1.  **음료 온도**: 현재 음료의 온도 (예: 갓 내린 커피 85°C)
2.  **환경 온도**: 주변 실내 온도 (예: 25°C)
3.  **냉각 계수 ($k$)**: 용기의 보온 성능 (아래 참고표 확인)
4.  **적정 온도**: 마시고 싶은 목표 온도 (예: 60°C)

### 💡 냉각 계수 ($k$) 참고 가이드

입력 시 참고할 수 있도록 프로그램 내에서도 안내되지만, 대략적인 값은 다음과 같습니다.

| 용기 종류           | 추천 $k$ 값      | 특징                                |
| :------------------ | :--------------- | :---------------------------------- |
| **머그컵 (뚜껑 X)** | `0.03` ~ `0.05`  | 열이 빠르게 식음                    |
| **종이컵 (뚜껑 O)** | `0.02` ~ `0.03`  | 뚜껑 덕분에 조금 더 천천히 식음     |
| **보온 텀블러**     | `0.005` ~ `0.01` | 진공 단열로 온도가 매우 오래 유지됨 |
